# Riot XMPP (unofficial)

[![npm](https://img.shields.io/npm/v/@valpro-labs/riot-xmpp.svg)](https://www.npmjs.com/package/@valpro-labs/riot-xmpp)

A typed TypeScript client for Riot XMPP. This library provides easy access to Riot XMPP with full TypeScript support.

## Installation

```bash
npm install @valpro-labs/riot-xmpp
```
